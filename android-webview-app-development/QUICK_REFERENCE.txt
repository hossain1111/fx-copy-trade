│ 🐛 TROUBLESHOOTING                                                         │
└────────────────────────────────────────────────────────────────────────────┘

"Cannot resolve symbol R":
  → Build → Clean Project
  → Build → Rebuild Project

Gradle sync failed:
  → File → Sync Project with Gradle Files
  → Check internet connection

SDK not found:
  → Tools → SDK Manager
  → Install API 21-34

WebView not loading:
  → Check INTERNET permission in manifest
  → Verify usesCleartextTraffic="true"

App crashes:
  → Check Logcat for errors
  → Verify all activities in manifest

Build error:
  → Clean project
  → Invalidate caches: File → Invalidate Caches / Restart
