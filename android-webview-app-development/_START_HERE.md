## 🐛 Common Issues (Quick Fixes)

### "Cannot resolve symbol R"
```
Fix: Build → Clean Project → Rebuild Project
```

### Gradle sync failed
```
Fix: File → Sync Project with Gradle Files

### WebView not loading
```
Fix: Check AndroidManifest.xml has INTERNET permission
     Verify usesCleartextTraffic="true"

Fix: Check Logcat for errors
